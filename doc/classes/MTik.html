Public Class methods

command(args)

An all-in-one function to instantiate, connect, send one or more commands, retrieve the response(s), close the connection, and return the response(s).

WARNING :If you use this call with an API command like

/tool/fetch it will forever keep reading replies, never returning. So do NOT use this with any API command that does not complete with a “!done” with no additional interaction.
